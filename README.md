# 🍅 Tomato - Food Delivery App

Tomato is a modern full-stack food delivery web application built using the MERN stack. It provides a seamless and responsive user experience for browsing food items, managing carts, and placing orders with integrated online payment.

---

## 🚀 Features

* 🍔 Browse food items by category
* 🔍 Smart filtering system
* 🛒 Add to cart functionality
* 💳 Online payment integration (Razorpay)
* 📦 Order management system
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Context API

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Payment:

* Razorpay

---

## 📂 Project Structure

client/ → React frontend
server/ → Backend API

---

## ⚙️ Installation

Clone the repository:

git clone https://github.com/ManavDobariya/Tomato.git

Install dependencies:

Frontend:
cd client
npm install
npm run dev

Backend:
cd server
npm install
npm start

---

## 🔐 Environment Variables

Create a `.env` file in the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
